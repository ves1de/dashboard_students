# Dashboard_Students

Проект разработан студентами Государственного университета «Дубна» в рамках предмета Технологии Программирования (ТП)

## Установка и запуск

Для корректной работы проекта потребуется установленный Node.js.

### Шаги для запуска:

1. Перейдите в директорию `app`:
cd app
2. Установите необходимые зависимости:
npm install
3. Пересоберите модуль с базой данных `better-sqlite3`:
npm rebuild better-sqlite3
4. Запустите приложение:
npm start

После запуска приложение будет доступно локально (обычно по адресу http://localhost:3000 или указанному в настройках).
